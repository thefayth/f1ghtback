# thefaythai Companion Contract

`thefaythai.companion.v1` is a manual text export, not an integration or shared database.

The envelope contains the purpose, jurisdiction, guide ID and version, source-manifest version, `includesHiddenRecords:false`, `copiedByUser:true`, and draft status. The user may add one local note before download.

Nothing is sent automatically. Import into thefaythai requires a separate user-controlled copy or file selection. The contract does not authorize directory access, hidden-record access, ChatGPT submission, filing, publication, or silent synchronization.
