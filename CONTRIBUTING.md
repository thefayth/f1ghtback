# Contributing

Contributions should preserve the product's narrow safety boundary:

- use official court or government sources;
- keep California and Utah workflows separate;
- route cross-state uncertainty to human review;
- collect no free-text case narrative or identifying detail;
- never claim legal advice, representation, filing, or deadline calculation;
- add tests for every source, jurisdiction, and boundary change.

Run `npm test` and `npm run lint` before opening a pull request. Do not include private records or realistic case fixtures in tests.
